Formato esperado del PDF:

  • El PDF debe tener texto seleccionable o una tabla real. Si el PDF es una imagen escaneada sin OCR, el sistema no podrá leer los criterios.
  • La tabla principal debe tener una columna con encabezado que contenga "criterio", "indicador", "variable" o "item" (case-insensitive).
  • Los criterios deben estar en filas debajo del encabezado. Si la tabla se repite en varias páginas, se tomarán las filas válidas y se eliminarán duplicados.
  • El encabezado puede estar en la primera fila o dentro de las primeras filas del PDF. También se aceptan tablas con encabezados como "Criterio de evaluación", "Indicadores y/o variables" o similar.
  • Si el PDF no contiene tabla, puede funcionar cuando los criterios aparecen como texto enumerado bajo una sección de criterios, por ejemplo: "1. Cumple con...", "2. Presenta...", etc.
  • Opcional: columnas "etapa" o "fase" para asignar etapa; columnas "comentarios", "descripción", "observaciones" o "desc" para guardar detalles del item; columnas "puntaje", "puntos" o "calificación" para definir el puntaje máximo.
  • No se recomienda usar PDFs con columnas fusionadas, criterios separados por saltos de línea sin tabla o imágenes insertadas, porque pueden no detectarse correctamente.
